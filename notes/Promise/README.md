# Promise 相关资源

## 基础

-   概念

    Promises象征着一个异步操作的最终结果。

-   三种状态

    ```
    pending                 初始状态
    fulfilled(resolved)     成功执行
    rejected                执行出错
    ```

    pending 状态的 Promise 对象可以转换到其它两种状态。
